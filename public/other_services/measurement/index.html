Measurement

This service is useful for measure the execution time of a method by create the execution scopes and checkpoints in your code, and then print the details of execution time step by step to determine where is the slowly code. You can use it with new transient instance to meansure inside a class, method, or use dependency injection pattern to meansure cross services.

    ITimeTracker timeTracker = new TimeTracker();
    using (timeTracker.BeginScope("Test"))
    {
        // Do something
        await Task.Delay(12);
        timeTracker.Checkpoint("Checkpoint 0");

        using (timeTracker.BeginScope("Inner Test"))
        {
            // Do something
            await Task.Delay(15);

            using (timeTracker.BeginScope("Inner Inner Test"))
            {
                // Do something
                timeTracker.Checkpoint("Checkpoint 1.1");
                await Task.Delay(20);
                timeTracker.Checkpoint("Checkpoint 1.2");
            }
        }
        timeTracker.Checkpoint("Checkpoint 1");
        using (timeTracker.BeginScope("Inner Test 2"))
        {
            // Do something
            await Task.Delay(10);
            timeTracker.Checkpoint("Checkpoint 1.3");
            timeTracker.Checkpoint("Checkpoint 1.4");
        }
    }
    Console.WriteLine(timeTracker.ToString());
